Manual: Instalación de Servidor de Correo

1. Actualizar el sistema

Primero actualizamos los paquetes del sistema para evitar errores.

sudo apt update

Actualiza la lista de paquetes disponibles.

sudo apt upgrade

Instala las últimas versiones de los paquetes.

sudo apt install apache2

Instala el servidor web Apache.

sudo apt install mysql-server

Instala el servidor de base de datos MySQL.

sudo apt install php php-mysql php-intl php-xml php-mbstring phjp-curl php-zip -y

Instala PHP y extensiones necesarias para aplicaciones web.

sudo apt update && sudo apt upgrade -y

Vuelve a asegurar que todo está actualizado.

2. Instalar Postfix (envío de correos)

sudo apt install postfix -y

Instala el servidor de correo Postfix.

sudo nano /etc/postfix/main.cf

Edita el archivo de configuración principal.

mynetworks = 192.168.1.0/24 127.0.0.0/8
myhostname = correu.caparrella.local
mydomain = caparrella.local
myorigin = $mydomain
mydestination = $myhostname, localhost.$mydomain, localhost, $mydomain
home_mailbox = Maildir/

Configuración básica del dominio y red local.

sudo service postfix restart

Reinicia el servicio para aplicar cambios.

3. Instalar Dovecot (recepción de correos)

sudo apt install dovecot-imapd dovecot-pop3d -y

Instala Dovecot para recibir correos.

sudo nano /etc/dovecot/conf.d/10-mail.conf

Edita la ubicación del correo.

mail_location = maildir:~/Maildir
sudo nano /etc/dovecot/conf.d/10-auth.conf

Configura la autenticación.

disable_plaintext_auth = no
auth_mechanisms = plain login

sudo service dovecot restart

Reinicia Dovecot.

4. Instalar Roundcube (webmail)

sudo apt install roundcube roundcube-core roundcube-mysql -y

Instala el cliente web de correo.

sudo nano /etc/roundcube/config.inc.php
$config['smtp_host'] = 'localhost:25';
$config['smtp_user'] = '';
$config['smtp_pass'] = '';
$config['mail_domain'] = 'caparrella.local';

Configura el envío de correos desde Roundcube.

5. Configurar Apache

sudo nano /etc/apache2/conf-available/roundcube.conf
Alias /roundcube /var/lib/roundcube

<Directory /var/lib/roundcube>
    Options FollowSymLinks
    AllowOverride All
    Require all granted
</Directory>

Permite acceder a Roundcube desde el navegador.

sudo a2enconf roundcube

Activa la configuración.

sudo service apache2 restart

Reinicia Apache.

6. Crear usuarios de correo

sudo adduser batman

Crea el usuario.

sudo maildirmake.dovecot /home/batman/Maildir

Crea el buzón de correo.

sudo chown -R batman:batman /home/batman/maildir

Asigna permisos.


sudo adduser robin
sudo maildirmake.dovecot /home/robin/Maildir
sudo chown -R robin:robin /home/robin/maildir

7. Enviar correo de prueba

telnet localhost 25

Conecta con el servidor de correo.

EHLO localhost
MAIL FROM:<batman@caparrella.local>
RCPT TO:<robin@caparrella.local>
DATA
Subject: Prova de correu

Bon dia! Això és una prova
.
QUIT

Envía un correo manualmente para comprobar que funciona.
